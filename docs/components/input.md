# 选项

#### 1. disabled
表示是否禁用状态，默认为 false，禁用。
#### 2. readonly
表示只读状态， 与 disabled 相似。但是 disabled 不可以被 Tab 键选中，<font color="#3eaf7c">readonly</font> 可以被选中，不能修改。
#### 3. error
给 Input 添加 <font color="#3eaf7c">error</font> 属性，会在该 input 右侧显示一个提示图标与文字，用于提示。
